Detailed Description

The implementation is done as part of the porting layer. All implementation is forwarded to the private Impl class. Coding this inline should speed-up code and reduce code size.

Member Enumeration Documentation

enum Priority
Enumerator
PRIO_LOW 

Priority below normal. Such a thread should only run if no other threads can run at that moment.

PRIO_NORMAL 

Normal priority, at which threads run that would otherwise not have their priority set. Typically the same prority as that of the main() thread.

PRIO_HIGH 

Priority above normal. Such a thread should run if possible (i.e. if not blocked), unless a higher priority thread can run.

PRIO_HIGHEST 

Highest priority. Such a thread should run at all times if it can (i.e. if not blocked).
